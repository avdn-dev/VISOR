name: British English in commits
description: User requires British English spelling in all commit messages — no Americanised English
type: feedback
---

Use British English in commit messages (e.g., initialiser not initializer, synthesise not synthesize, optimise not optimize, colour not color).

**Why:** Strong user preference — Americanised spelling is unacceptable.

**How to apply:** Every commit message must use British spelling. Double-check words like -ise/-ize, -our/-or, -ised/-ized before committing.
